# GraphQL Hands-On

* React + Apollo Client
* NodeJS GraphQL-Server with WebSocket-Subscriptions
* PouchDB to CouchDB Data-Synchronization

## GraphQL-Server

```bash
npm start
```

* [GraphiQL](http://localhost:8080/graphiql)

## GraphQL-Client

```bash
npm run dev
```

* [React + Apollo](http://localhost:9090)

## Initial data import

### Production Environment

```bash
npm run import
```
